Album

나의 추억들이 담긴 곳에서

Memory #1 : 후쿠오카 타워

차가운 겨울바람을 맞으며 찾았던 후쿠오카
유후인 기찻길이 제일 기억에 남는다. 가장 재밌었던 것은 길거리 먹방

Memory #2 : 도쿄

이색적인 경험 많은 등불들이 날라가고 소원을 비는게 신기했다
여러개의 분홍빛 등불들이 밤하늘과 잘 어울렸다

Memory #3 : 활화산

실제로 활동 중인 화산을 본 건 처음이라 신기했고, 동시에 약간의 긴장감도 들었다
자연의 힘이 얼마나 거대한지 실감할 수 있는 경험이었다

Memory #4 : 겨울바다

겨울바다라 그런지 사람이 거의 없었다
바람이 세고 차가웠지만, 바다 풍경은 맑고 깨끗했다
한적해서 조용히 걷기 좋았고, 생각 정리하기에 좋은 장소였다
모래알이 정말 부드러웠던게 기억에 남는다

Memory #5 : 이스탄불 모스크

규모가 예상보다 훨씬 컸고, 건축 양식이 독특했다
내부는 조용하고 엄숙한 분위기였고, 관광객보다 기도하러 온 현지인들이 인상 깊었다
종교적 의미를 넘어, 문화와 역사에 대한 이해가 조금 더 깊어지는 경험이었다 야경이 정말 이뻤다

Memory #6 : 하기아 소피아 맨션 수영장

술탄아흐메트 중심에 위치한 힐튼 큐리오 컬렉션 호텔의 지하 수영장
오스만 시대 건축 양식을 그대로 살린 석조 천장과 기둥이 인상 깊었다
역사적인 공간 안에서 수영을 할 수 있다는 점이 신기했다

Memory #7 : 무지개 해안도로

진짜 엄청 추운데 사진 찍을려고 줄 서있던 기억이 있다

Memory #8 : 구름이

박스안에 들어가서 저런 표정을 하고 있는게 너무 귀엽다 다시봐도 표정이 너무 웃기다

Memory #9 : 롯데타워

롯데타워에서 내려다 봤을 때다 뭔가 빌라만 가득보여서 그닥 이쁘진 않은 것 같다

Memory #10 : 바다낚시

처음으로 친구들이랑 저녁에 바다낚시 하러 갔을 때다 이때가 정말 재밌었는데 그립다 물고기는 한마리도 못잡고 꽃게만 잡은게 기억이 난다

Memory #11 : 스키

저녁에 스키 타러 갔는데 사람이 거의 없었던게 생각이난다
바로바로 스키탈 수 있어서 정말 좋았는데 고급코스에서 타나가 넘어져서 엉덩이에 멍든게 생각이난다

Memory #12 : 노형수퍼마켙

이름만 수퍼마켙이지 실제로는 대형 미디어 테마파크다
레트로 감성도 있고 다양하다 내가 사진찍은곳 이름이 기억이 잘 안나네...
생각보다 사진 찍을 수 있는곳이 엄청 많은 곳이다
